# Compliance models

These models are archived from the monolith.

They are **not** part of the runtime system.

Only activate if:

- required for regulation, or
- actively used in features.

## Strategy

| Domain    | Status        |
| --------- | ------------- |
| Listings  | Production    |
| Bookings  | Production    |
| Search    | Production    |
| Checkout  | Production    |
| Users     | Modular       |
| Compliance | **Frozen** 🧊 |

This matches how larger platforms scope risk: keep revenue paths hot; park low-ROI surface area until there is a concrete regulatory or product need.
