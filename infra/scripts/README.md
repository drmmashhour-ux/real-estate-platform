# Scripts

Development and deployment scripts.

- **Database**: From repo root run `npm run db:push` (apps/web-app) or `npm run db:seed`.
- **Migrations**: `npm run db:migrate` in the workspace that owns the Prisma schema (e.g. apps/web-app).

Add project-specific scripts here (e.g. seed-bnhub.ts, backup-db.sh).
