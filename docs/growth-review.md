# Weekly growth review

**30–45 minutes**, same slot every week (e.g. Monday 10:00). Owner: **Founder** or **Growth lead**. Output: **3 bullets** posted to team channel.

---

## 1. Numbers (5 min)

Pull from **`/admin/early-users`**, analytics, and **`7day-metrics.md`** / sheet.

| Metric | This week | Last week |
|--------|-----------|-----------|
| Outreach sent | | |
| Replies | | |
| Signups | | |
| Active users | | |
| Landing leads (`/early-access`) | | |
| Referrals (`Referral` rows / usedAt) | | |

**Rates**

- Message → reply %  
- Reply → signup %  
- Signup → active %  

---

## 2. What worked (10 min)

- Best **channel** (DM vs TikTok vs FB vs partner).  
- Best **script** or **hook** (qualitative + any spike in signups).  
- Best **content** piece (views → clicks if tracked).

**Decision:** **Scale** one channel next week (+20% effort or budget).

---

## 3. What failed (10 min)

- Channels or messages with **near-zero** reply.  
- Drop-offs in product (where users quit in analytics).

**Decision:** **Pause** or **rewrite** one thing; assign owner.

---

## 4. Next week (10 min)

- **Top 3 bets** (e.g. “50 DMs to FB group members”, “3 partnership calls”, “fix checkout step 2”).  
- **Owner + date** for each.

---

## Honesty rule

If data is missing, write **“instrument this week”** and add one tracking task—don’t fake benchmarks.
