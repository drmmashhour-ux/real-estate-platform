# Exercise 4 — Final Teacher-Required Workflow

This document captures the **exact workflow** expected for Exercise 4.

## Objective

Recreate Exercise 4 using:

- Shapes (Ellipse, basic shapes)
- Shape Builder Tool (`Shift + M`)
- Visible construction logic

Do **not** use:

- Raster artwork
- Image Trace
- Freehand "final drawing" with the Pen Tool

---

## Step 1 — File setup (mandatory)

1. Create a new file: **11" x 8.5"** (RGB).
2. Create layers:
   - `REFERENCE` (locked)
   - `CONSTRUCTION` (optional, but strongly recommended)
   - `FINAL`
3. Place the template in `REFERENCE`.
4. Set template opacity to **20-30%**.
5. Lock `REFERENCE`.

---

## Step 2 — Build using only simple shapes

Start only with:

- Circles (Ellipse Tool)
- Basic geometric shapes

Do **not** draw final silhouettes directly.

---

## Step 3 — Orbit lines

1. Create multiple circles.
2. Duplicate circles (`Alt + drag`).
3. Resize as needed.
4. Apply stroke:
   - **Purple**
   - **No fill**

These must remain clean vector strokes.

---

## Step 4 — Body (from shapes)

1. Create one circle.
2. Use it as the bird body base.

---

## Step 5 — Wings (Shape Builder required)

1. Create **2-3 overlapping ellipses**.
2. Select all wing shapes.
3. Activate Shape Builder (`Shift + M`).
4. Drag across regions to **combine**.
5. Hold `Alt` and click/drag to **remove** unwanted regions.

This construction logic must remain visible in your structure.  
The teacher checks this specifically.

---

## Step 6 — Beak (no freehand)

1. Create either:
   - A triangle, or
   - A stretched ellipse
2. Use Shape Builder to cut/merge into final beak geometry.

---

## Step 7 — Eye

1. Create a small circle.
2. Apply a simple fill.

---

## Step 8 — Motion curves

1. Draw full circles.
2. Use the Scissors Tool (`C`) to cut circles into arcs.

Do **not** draw arcs manually.

---

## Step 9 — Color application

Apply colors **after** all shapes are constructed.

You may either:

- Fill regions via Shape Builder, or
- Select finalized shapes and apply fills directly

---

## Step 10 — Keep construction evidence (important)

Do not delete all construction work.

Keep at least one of the following:

- Some overlapping source shapes, or
- A duplicated version of the artwork with visible construction

Recommended: keep one "construction" version and one "clean" version.

---

## Step 11 — Final clean version

On `FINAL` layer:

- Keep only clean, resolved shapes
- Remove unintended overlaps
- Apply proper fills

---

## Step 12 — Validation checklist (critical)

Before submitting, confirm:

- [ ] Clicking any object shows editable anchor points
- [ ] No embedded raster image is used for the final drawing
- [ ] Shapes are not freehand-built as final forms
- [ ] Shape Builder construction logic is evident
- [ ] Artwork is cleanly grouped and editable

---

## Step 13 — Save

File name format:

`Ex_4_YourName.ai`

---

## What the teacher will check

When opening the file, they will verify:

1. Is it built from shapes?
2. Was Shape Builder used?
3. Is every part editable?
4. Is it true vector (not image-based)?

Passing depends on all four being clearly true.

---

## Key mindset

Build messy with shapes first, then clean with Shape Builder.

Do not draw the final form directly.
