# Revenue plan + conversation scripts (senior living)

Simple positioning: you sell **outcomes** (visits → residents → revenue), not “software.”

---

## 1. Revenue plan (simple + high conversion)

### Phase 1 — First 10–50 operators

Keep it **very simple:**

| Element | Detail |
|--------|--------|
| Onboarding | **FREE** |
| First leads | **FREE** (first **2–3** qualified leads) |
| Then | **$25–$50** per **qualified lead** |

Define “qualified lead” in **one sentence** on paper/email so disputes stay rare.

---

### Phase 2 — Once validated

Add:

**Option A — Pay per lead**

- **$30–$80** per lead (tier by **quality** / geography / urgency)

**Option B — Subscription**

- **$150–$300 / month**
- Includes **X** leads (spell out **X** and rollover rules)

---

### Phase 3 — Scale

Premium add-ons:

- **Featured placement**
- **Priority ranking**
- **AI-optimized exposure** (only promise what the product delivers)

---

### Key rule

- 👉 You don’t sell **“software”**
- 👉 You sell **visits → residents → revenue**

---

## 2. Message: visit → signed contract

### After the visit (critical moment)

```
Hi 😊 just checking in — how did the visit go?

Most families tell me this is when things become much clearer.

If you liked the place, the next step is usually to secure availability — I can guide you through it if you want.
```

---

### If they liked it

```
That’s great to hear 🙏

Places like this tend to fill quickly, especially at this price range.

If you feel it’s a good fit, I recommend moving forward while it’s still available — I can help you with the next step.
```

---

### Soft close (no pressure)

```
No pressure at all — it’s an important decision.

But if it felt right during the visit, it’s usually a good sign.

Would you like me to help you move forward or explore one more option before deciding?
```

---

## 3. Operator follow-up (close deals faster)

### After sending them a lead — call script

**Open:**

> “Hi [Name], just wanted to follow up on the **family we sent you**. Did you manage to **speak with them** or **schedule a visit**?”

→ Let them answer.

**If no contact yet:**

> “If not, I recommend **reaching out quickly** — they’re actively looking and usually decide fast.”

**If visit scheduled:**

> “Perfect — that’s exactly what we want. After the visit, a **quick follow-up** usually makes a big difference.”

**If they didn’t respond to the family:**

> “This is actually where **most opportunities are lost** — fast response increases conversion a lot. If you want, I can help you with a **simple follow-up message**.”

**Close:**

> “Our goal is to help you **convert** these leads, not just send them.”

## 4. Real conversation (step-by-step)

### Example 1 — Family flow

**User:** “I’m looking for a place for my mother in Laval”

**You:**

```
I understand 🙏 I can help you with that.

Can you tell me:
• does she need some help daily or is she mostly independent?
• and roughly your budget?
```

**User replies.**

**You:**

```
Thank you — based on that, I found a few good options.

This one is usually a strong match for families in your situation 👇
```

👉 Send **3 options** · 👉 **Highlight 1**

**User:** “Looks good”

**You:**

```
Great 😊

The best next step is to visit — it really helps you feel if it’s the right place.

Would you like me to help you arrange a visit?
```

---

### Example 2 — Hesitation

**User:** “I’m not sure yet”

**You:**

```
That’s completely normal — it’s a big decision.

If it helps, we can start with one visit just to get a better idea — no commitment at all.
```

---

### Example 3 — Ready to move

**User:** “We liked it”

**You:**

```
That’s great 🙏

If it felt right during the visit, the next step is usually to secure the spot while it’s still available.

I can help guide you through it if you’d like.
```

---

## Key sales psychology

- You **guide**, you don’t push  
- You **reduce uncertainty**  
- You create **urgency** (gently)  
- You make the **next step obvious**
