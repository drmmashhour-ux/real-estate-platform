# Outreach tracker template

Copy this table into **Notion**, **Excel**, **Google Sheets**, or **Airtable**.

## Columns

| Column | Description |
|--------|-------------|
| **Name** | Contact first and last name. |
| **Firm** | Fund, angel group, or independent. |
| **Email** | Primary email (verify before sending). |
| **LinkedIn** | Profile URL. |
| **Tier** | 1 / 2 / 3 — see **investor-targeting.md**. |
| **Status** | `Not contacted` / `Contacted` / `Replied` / `Meeting` / `Passed` / `Paused` |
| **Last contact date** | Date of last outbound (or last inbound if you track replies here). |
| **Next follow-up date** | When the next touch is due (if none, leave blank or “—”). |
| **Notes** | Thesis fit, portfolio refs, what they said, intro links. |

## Usage

1. **Add** one row per qualified investor.
2. **Update status** after every send or reply.
3. **Set next follow-up** when you send — use **follow-up-system.md** and **outreach-strategy.md** for timing.
4. **Weekly review** — move stale “Contacted” rows forward or mark Passed.

**Privacy:** Do not share this sheet publicly; treat emails as personal data.
