# Outreach playbook — LECIPM Manager

## 1. Daily workflow

- **Research & capture:** Add **10–20** new investor rows (fund, partner, angel) with source and thesis note.  
- **Send:** **10 personalized** outbound messages (email or warm intro request) — personalization on **line one** only; body stays tight.  
- **Track:** Log send date, channel, and template variant in CRM; note any reply same day.

## 2. Weekly

- **Pipeline review:** Sort by **next follow-up** and **status**; clear stale dates.  
- **Warm bias:** Re-engage referrals and intro paths before scaling cold volume.  
- **Refine:** Drop one low-performing subject line; keep one angle that got replies (internal learning only — do not publish as statistics).

## 3. Goals (internal targets — not external claims)

| Metric | Target |
|--------|--------|
| Investors touched / week | **50–100** (team-dependent; do not sacrifice quality) |
| Reply rate | **10–20%** internal aim — varies widely by list quality |
| Meetings / month | **5–10** internal aim — optimize scheduling links and intro leverage |

*These are planning numbers, not traction disclosures.*

## 4. Best practices

- **Personalize the first line** — portfolio company, thesis post, or mutual connection.  
- **Keep emails short** — busy investors scan; respect their time.  
- **Always include a CTA** — one ask: short call or calendar link.  
- **LECIPM Manager** naming consistent; **soft launch ready** only if true.  
- **Never** invent users, revenue, or growth rates in outreach.

## 5. Conversion focus

- **Friction:** Offer deck async + optional call.  
- **Credibility:** Point to product demo or `/investors` page if public — not unverified metrics.  
- **Follow-through:** Book time in the same thread when they reply.
