# Daily execution plan

## Daily routine

| Step | Action |
|------|--------|
| 1 | **Identify** 10–20 new qualified investors (or fewer if research-heavy Tier 1). |
| 2 | **Send** first-touch outreach (email and/or LinkedIn per your strategy). |
| 3 | **Update** the outreach tracker — status, last contact date, next follow-up. |
| 4 | **Send** due follow-ups for existing threads (from **follow-up-system.md**). |

**Block 30–60 minutes** on the calendar if you’re doing this consistently — same time daily helps.

---

## Weekly routine

| Step | Action |
|------|--------|
| 1 | **Review** all replies — move meetings to calendar; update Passed / Paused. |
| 2 | **Adjust** messaging — which angle (**messaging-angles.md**) got replies or meetings? |
| 3 | **Refine** templates — one change at a time so you know what improved. |
| 4 | **Clean** stale rows — close loops with final follow-ups where needed. |

---

## Guardrails

- **No fabricated metrics** in any outbound or reply.
- **Verify** links before weekly sends.
- **Tier 1** gets research time before first touch; Tier 3 should not consume the same time per contact.

---

## Outcome

Structured repetition → more conversations → clearer feedback on what resonates. Volume without quality burns reputation; quality without volume slows learning — balance using tiers.
