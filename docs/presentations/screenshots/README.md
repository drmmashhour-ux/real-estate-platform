# Pitch deck product screenshots

Place **PNG** files here (recommended **1280×800** or wider). The build script embeds:

| File | Suggested capture |
|------|-------------------|
| `homepage.png` | Marketing or app home |
| `listing.png` | Property / stay listing detail |
| `map_search.png` | Map or search results |
| `dashboard.png` | Host or broker dashboard |

If a file is **missing**, the builder generates a **dark placeholder** with a gold frame — replace with real captures before investor meetings.

**Capture tip:** Run the web app locally, use browser devtools device frame or full-width desktop, export PNG.
