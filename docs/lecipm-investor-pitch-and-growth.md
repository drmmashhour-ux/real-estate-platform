## 2. Marketing system (growth engine)

### Funnel

1. **Hook (ad)** — “Your listing might be losing buyers”  
2. **Landing** — “Check your listing score”  
3. **Input** — Paste listing **or** create listing  
4. **Result** — Show trust score + issues  
5. **Conversion** — “Fix and get verified”  

### Ads — Google

Keywords (examples):

- “sell house faster”  
- “real estate listing tips”  

### Ads — Meta

Video narrative: bad listing → low trust → fixed listing → verified  

### Viral loop

User sees score → improves listing → better results → shares

## 3. Pixel-level UI (key screens)

### Listing page

- **Top:** Title + price  
- **Right panel:** Trust score (large), badge (gold), readiness bar  
- **Below:** Missing items, warnings, passed checks  
- **CTA:** Fix now  

### Dashboard

- **Grid:** Trust score, listings verified, issues, leads  
- **Table:** Listings  

### Copilot

- **Bottom-right floating panel:** input + quick actions  
- **Suggestions:** “Find deals” · “Analyze this” · “Why not selling?”
