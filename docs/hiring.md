# Hiring system

Practical hiring for a **small, execution-heavy** team. Customize titles and comp to your jurisdiction and stage.

---

## Role templates (job descriptions — shorten for posts)

### Full-stack engineer
- **You will:** Build and ship Next.js/TypeScript features, APIs, and data paths in a monorepo; own quality and performance with the team.  
- **You have:** 3+ years shipping web apps; SQL/Postgres or Prisma comfort; Git + code review habits.  
- **Nice:** Real estate, payments, or maps experience.

### Mobile engineer
- **You will:** Own React Native / Expo app quality, releases, and parity with core BNHub flows.  
- **You have:** Shipped apps to stores; API integration experience.

### UI/UX designer
- **You will:** End-to-end product design for marketplace and admin tools; work in Figma; validate on staging.  
- **You have:** Portfolio with complex flows (booking, forms, dashboards).

### Growth / marketing
- **You will:** Run content + paid experiments; own landing pages and attribution with engineering support.  
- **You have:** B2C or marketplace growth experience; comfort with numbers.

### Operations / sales
- **You will:** Onboard hosts/brokers; run demos; maintain CRM; feed product insights.  
- **You have:** SaaS or marketplace ops; French/English if you serve Québec.

---

## Hiring script (first conversation — 20 min)

1. **Context (2 min):** What BNHub/LECIPM does; stage of company; this role’s **one** main outcome in 90 days.  
2. **Their story (5 min):** Last win; how they like to work; remote/async comfort.  
3. **Depth (10 min):** 2–3 role-specific questions (see checklist below).  
4. **Their questions (3 min):** Roadmap, team, comp range band (optional).

---

## Interview process (default)

| Round | Goal |
|-------|------|
| **Screen** | Culture fit, clarity, basic skill signal (30 min). |
| **Skills** | Take-home **or** live exercise **≤ 2h** total—respect candidate time. |
| **Team** | Cross-functional chat (eng + product/growth) (45 min). |
| **Founder** | Values, urgency, compensation (30 min). |

**Decision within 5 business days**; one clear “yes/no/next” to candidate.

---

## Evaluation checklist

**All roles**

- [ ] Clear written and spoken communication  
- [ ] Evidence of **shipping** (not only planning)  
- [ ] Handles ambiguity without stalling  

**Engineering**

- [ ] Reads unfamiliar code and proposes a sane change  
- [ ] Explains tradeoffs (speed vs quality)  
- [ ] Security awareness (secrets, auth basics)  

**Design**

- [ ] Walks through a past project: problem → iteration → result  
- [ ] Can critique our staging UI constructively  

**Growth**

- [ ] Describes one experiment with metric and outcome  
- [ ] Channel thinking (not only “post more”)  

**Ops / sales**

- [ ] Role-play short discovery call  
- [ ] CRM discipline and follow-up habit
