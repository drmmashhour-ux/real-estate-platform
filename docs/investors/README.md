## Consistency rules

1. **Naming:** Use **LECIPM Manager** everywhere in this package (not mixed with other product names unless explicitly cross-referencing the broader platform).
2. **Tagline family:** **AI-managed real estate & stays marketplace** (or close variants listed in each doc).
3. **No fake traction:** Do not present fabricated MAU, GMV, revenue, or growth rates. **Traction** = **platform built** and **soft launch ready** unless you replace copy with **verified** live metrics.
4. **Numbers:** Treat all financial figures in [financial-projections.md](./financial-projections.md) as **illustrative** unless you substitute **audited** or **actual** data.
5. **Revenue streams:** **Booking fees** (take on completed stays), **insurance / partner leads**, **checkout upsells / add-ons** — as described in the one-pager and deck.
