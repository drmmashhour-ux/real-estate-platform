# LECIPM — First 100 Users Playbook

**Goal:** ~100 **intent-rich** accounts across five segments (**20 each**). Quality over volume—adjust totals if one segment saturates faster.

**Tracking:** spreadsheet columns: `email`, `segment`, `source`, `invite_date`, `activated_y/n`, `activation_action`, `paid_y/n`, `notes`.

---

## Segment allocation

| Segment | Count | Definition of “counts” |
|---------|-------|-------------------------|
| Brokers | 20 | Completed signup + **broker dashboard accessed** + **CRM or lead flow touched** |
| Hosts | 20 | BNHub listing **published** OR host dashboard **listing draft ≥80%** + **calendar/contact path** |
| Sellers | 20 | FSBO listing **ACTIVE** OR wizard completed with **pending review** |
| Buyers / renters | 20 | Saved search OR saved listing OR contact CTA submitted |
| Investors / serious users | 20 | Investor dashboard opened + **scenario or portfolio interaction** OR explicit demo completed |

---

## Brokers (20)

| Field | Content |
|-------|---------|
| **Offer** | **90-day reduced lead-unlock friction**: priority onboarding call + template scripts for replies (Concierge). Not a fee discount unless commercially approved—prefer **time + setup** value. |
| **Hook** | “See qualified inquiry flow **without** juggling five tabs.” |
| **Outreach channel** | LinkedIn DM + warm intro from mortgage/real estate circles; **QC brokerage meetups**. |
| **Message angle** | Speed to contact + **audit trail** on platform (no clinical claims—professional ops). |
| **Conversion step** | Book **20-min Zoom** → live account creation → **unlock test lead** (sandbox or micro-market). |
| **Activation action** | **Unlock one lead** OR **add three contacts** to CRM pipeline in-app. |

---

## Hosts (20)

| Field | Content |
|-------|---------|
| **Offer** | **Listing setup concierge**: photo checklist, pricing sanity pass, BNHub wizard walkthrough. |
| **Hook** | “Premium stays deserve **premium ops**—one dashboard for bookings and payouts direction.” |
| **Outreach channel** | Instagram DM to boutique hosts; existing landlord networks; **referral from one seller**. |
| **Message angle** | Control + transparency + **mobile visibility** for bookings. |
| **Conversion step** | Calendar link → **account + draft listing** same session. |
| **Activation action** | **Publish** listing OR complete **pricing + availability** with **≥5 photos**. |

---

## Sellers (20)

| Field | Content |
|-------|---------|
| **Offer** | **FSBO wizard assisted session** + compliance copy review (informational—not legal advice). |
| **Hook** | “List with **clarity**—buyer-ready detail page, not a PDF graveyard.” |
| **Outreach channel** | Local Facebook groups (luxury housing), realtor-adjacent newsletters, **seller seminars**. |
| **Message angle** | Exposure + professional presentation + **platform-supported** inquiry flow. |
| **Conversion step** | Lead magnet: **“Luxury listing checklist PDF”** → signup → wizard step 1 same day. |
| **Activation action** | Listing **submitted for review** or **ACTIVE**. |

---

## Buyers / renters (20)

| Field | Content |
|-------|---------|
| **Offer** | **Saved search + alert** setup done **for** them on first call (concierge). |
| **Hook** | “Search that respects **your** criteria—save, compare, act.” |
| **Outreach channel** | Content-led (see `growth-content-engine.md`); referrals from brokers; **open house QR**. |
| **Message angle** | Discovery without spam; **BNHub + residential** in one ecosystem. |
| **Conversion step** | **Save 3 listings** + enable notifications (email/push). |
| **Activation action** | **Contact seller/host** OR **book stay** OR **saved search active**. |

## Investors / serious users (20)

| Field | Content |
|-------|---------|
| **Offer** | **Private walkthrough** of investor dashboard + **how revenue hubs tie together** (platform narrative). |
| **Hook** | “Operating-system view of **marketplace economics** you can diligence.” |
| **Outreach channel** | Warm intros; **small angel syndicate**; family offices with real estate thesis. |
| **Message angle** | Governance + revenue visibility + **multi-hub** asset. |
| **Conversion step** | **Calendar demo** → account + **portfolio/scenario** interaction. |
| **Activation action** | Complete **one scenario path** OR **return session within 7 days**. |

## Onboarding sequence (all segments)

| Day | Action |
|-----|--------|
| **D0** | Welcome email: **one CTA** only (complete activation action). Include **human name** + reply-to. |
| **D1** | Short checklist email (3 bullets max) + link to **help** + **office hours**. |
| **D3** | If inactive: personal **plain-text** email from founder (“what blocked you?”). |
| **D7** | Segment-specific tip + **single metric** (“hosts who complete pricing convert 2×”). |
| **D14** | Ask for **feedback call** offer (15 min) **only** if user showed weak activation. |

---

## Manual concierge onboarding

**Rules**
1. **Every host + broker** in first 100 gets **one live touch** (Zoom or async Loom).
2. **Office hours:** **2× weekly** 45-min block; publish link in welcome email.
3. **Slack/WhatsApp cohort** (optional): **max 40 people** to avoid noise; archive at D90.

**Concierge script (open)**
> “I’m [Name]. You’re in the early cohort—my job is to remove friction. Your **one goal today** is [activation action]. If anything breaks, you have my direct line for **48h**.”

## First-value moments (design for these)

| Segment | First value moment |
|---------|-------------------|
| Broker | First **lead unlocked** or pipeline row moved |
| Host | First **inquiry or booking request** received |
| Seller | Listing **live** with shareable URL |
| Buyer | **Saved alert** fires with **≥1 match** |
| Investor | Dashboard shows **meaningful non-empty state** |

---

## Retention triggers

| Trigger | Action |
|---------|--------|
| **Saved search but no return D5** | Email: “3 new matches” **only if true** |
| **Host calendar gap** | Nudge to update availability |
| **Broker dormant D7** | Offer **import CSV** or template |
| **Payment dropped** | Recovery email + **support escalation** |

---

## Kill / pivot rules

- If segment **activation <15%** after **30 invites**, change **offer** or **ICP**—not ads budget.
- If **payment failures >5%** on segment, stop outbound until engineering sign-off.
