# LECIPM — Live Demo Script (3–5 min)

## 0) Hook (10–15s)

**Say:**

> LECIPM is an AI-powered real estate marketplace that doesn’t just list properties—it optimizes pricing, growth, and conversion automatically.

---

## 1) Entry & Scarcity (Homepage)

**Open:** `/en/ca`

**Show:** Launch banner + Hero

**Say:**

> From the first screen, we use real-time signals—early users and demand—to drive urgency and conversion.

**Point to:**

- Launch banner (“X spots left”)
- Hero CTAs

---

## 2) Smart Discovery (Search / Feed)

**Action:**

- Use Smart Search → type “Montreal”  
- **Or** open `/en/ca/feed`

**Say:**

> Discovery is personalized. We rank listings using demand, user behavior, and social proof.

**Point to:**

- Instant suggestions
- Feed cards (price, city, badges)

---

## 3) Social Proof + Trust

**Open:** `/listings` or a listing detail

**Say:**

> We surface real engagement signals—bookings, views—to build trust and increase conversion.

**Point to:**

- Green social proof bullets
- (If present) “Trusted listing / Top host”

---

## 4) AI Pricing (City + Listing)

**Open:**

- `/en/ca/dashboard/admin/market/city-pricing`  
- *(optional)* `/listing-pricing`

**Say:**

> Our AI analyzes demand and recommends pricing adjustments—city-level and listing-level.

**Point to:**

- Demand score
- “Increase price / Decrease price”
- Suggested %

---

## 5) Conversion Engine (Nudges)

**Open:** a listing page

**Say:**

> We predict user intent and show smart nudges—only when supported by real signals.

**Point to:**

- Nudge card (“Getting attention”, etc.)

---

## 6) Growth Engine (Broker Intelligence → Ads)

**Open:** Broker dashboard with `BrokerIntelligencePanel`

**Action:** Click “Generate Campaign”

**Say:**

> From insights to action in one click—we generate ad campaigns instantly.

**Point to:**

- TikTok / Meta / Google copy modal

---

## 7) Campaign Simulation + Optimization

**Open:** `/dashboard/broker/campaigns`

**Action:**

- Run a campaign (simulation)
- Click “Optimize Campaign” (dry run)

**Say:**

> We simulate performance and recommend what to do next—scale, pause, or improve copy.

**Point to:**

- CTR / conversions / spend
- Recommendation + reason

---

## 8) Marketplace Brain (Coordination)

**Open:** `/en/ca/dashboard/admin/ai-brain`

**Say:**

> All signals roll up into one decision layer—the Marketplace Brain—coordinating pricing, growth, and trust.

**Point to:**

- Top actions
- Priority badges

---

## 9) Launch Mode (Orchestration)

**Open:** `/en/ca/dashboard/admin/launch-control`

**Action:** Show status (running / day X)

**Say:**

> With one click, we orchestrate the launch—tracking progress, prioritizing actions, and aligning all systems.

---

## 10) Close (10–15s)

**Say:**

> LECIPM is a self-optimizing marketplace—learning from behavior, improving pricing and growth, and increasing conversion in real time.

---

## Key messages to emphasize

| Theme | Line |
|-------|------|
| Positioning | Not a listing site → an **optimization engine** |
| Loop | **Closed loop:** behavior → insights → actions → results |
| Safety | **Safe automation:** recommendations first, controlled execution |
| Trust | **Real signals only:** no fake urgency |

---

## Demo tips (so it never breaks)

- If an API is empty → say: **“This is live—signals populate as users interact.”**
- Keep **`dryRun = true`** for optimizer
- Have **one campaign pre-created** (for instant metrics)
- Keep **one listing with bookings > 10** for social proof

---

## 30-second backup pitch (if interrupted)

> We built an AI marketplace that automatically adjusts pricing, generates and optimizes campaigns, and increases conversion using real-time demand and user behavior. It’s not just a platform—it’s a system that grows itself.
