# First 100 users — lightweight metrics

**Purpose:** A simple scoreboard for manual or spreadsheet tracking. This is not a product analytics build.

## Core fields (weekly or daily row)

| Metric | Definition (keep consistent) |
|--------|-------------------------------|
| **Messages sent** | Outbound DMs or emails to owners (exclude internal). |
| **Replies received** | Any human reply (even “not interested”). |
| **Listings onboarded** | You have link/address + confirmed you’ll promote. |
| **Leads generated** | Agreed definition: e.g. qualified inquiry forwarded / viewing booked. |
| **Paid users** | Someone paid listing fee or per-lead (per your model). |
| **Revenue** | Cash collected in the period (match your books). |

## Optional columns

- Channel (Marketplace, TikTok, referral)
- Objection theme (price, trust, timing)

## Where this lives

- **Default:** spreadsheet or Notion — fastest.
- **Repo:** this file is the canonical *definition* of what to track; no heavy dashboard required.

## Internal page

If enabled, **`/admin/early-users`** includes a short **First 100 funnel** strip with links to this doc set and the public **`/get-leads`** page.
