# LECIPM — Ad image design specs (Canva-ready)

Professional, consistent assets for Meta (Feed / Reels) and Google Display. Build in Canva using exact hex values below.

---

## Global style

| Token | Value |
|--------|--------|
| Background | `#000000` |
| Primary (gold) | `#D4AF37` |
| Primary hover / brighter gold (optional accents) | `#E5C158` |
| Body text | `#FFFFFF` |
| Muted text | `#B0B0B0` |
| Style | Minimal, premium — generous padding, restrained copy |
| Imagery | Prefer **real UI screenshots** (dashboard, listing assistant, deal score); blur sensitive data |

**Fonts (suggestion):** Headlines — serif luxe (e.g. Cormorant / Playfair); UI / CTA — clean sans (Inter / DM Sans).

---

## Design 1 — Broker ad

**Objective:** Broker acquisition — demo bookings.

**Layout**

1. **Top:** Small uppercase label in gold — `For Brokers`
2. **Center (hero):** Large white headline —  
   **Close more deals with AI**
3. **Supporting line (optional, smaller, muted):**  
   *Leads, listings, and deal intelligence in one workflow.*
4. **Bottom:** Pill / button treatment (gold fill, black text):  
   **Book Demo**
5. **Optional:** Faint **dashboard or listing-assistant screenshot** behind text (opacity ~15–25%, slight blur).

**Safe zone:** Keep headline + CTA inside central 80% height on Reels (avoid bottom UI chrome).

---

## Design 2 — Property ad

**Objective:** Buyer / investor traffic — listing + AI hook.

**Layout**

1. **Full-bleed property photo** (hero exterior or bright interior).
2. **Overlay:** Bottom **linear gradient** black → transparent (approx. 60% height from bottom), opacity strong enough for white text.
3. **Overlay text (white, bold):**  
   **AI Analysis Available**
4. **Bottom CTA** (gold outline or solid gold button):  
   **View Now**

**Photo:** High contrast, well-lit; avoid heavy filters that clash with gold text.

---

## Design 3 — Deal score

**Objective:** Feature / proof — deal intelligence.

**Layout**

1. Solid **#000000** background.
2. **Center:** Large circular badge (gold stroke 2–4px, optional inner glow):  
   **`82`**  
   **`/100`** (smaller, same gold)
3. **Label under circle (gold, small caps):**  
   **Deal Score**
4. **Subhead (white):**  
   **Know when to act**

Optional: thin gold divider or subtle grid pattern at 5% opacity.

---

## Design 4 — Pain point

**Objective:** Pattern interrupt — broker pain.

**Layout**

1. **Big bold white headline (2 lines max):**  
   **Brokers are losing deals**
2. **Subtext (muted or gold):**  
   **LECIPM fixes this**
3. **CTA (gold button):**  
   **Start Now**

Keep copy short; no fine print on image (use ad platform fields for disclaimers).

---

## Export formats

| Use | Size | Notes |
|-----|------|--------|
| Instagram square | **1080 × 1080** | Feed, carousel |
| Instagram / TikTok / Reels vertical | **1080 × 1920** | Safe areas: top ~120px, bottom ~280px for UI / captions |
| Meta / Google Display | Generate **1:1** and **9:16** first; scale 1.91:1 or 16:9 from same master if needed |

**Export:** PNG or JPG, sRGB, max weight per platform limits; keep text layers editable in Canva source.
