# Deck screenshots

PNG files used by `scripts/build_bnhub_premium_deck.py`:

- `homepage.png` — home or marketing hero
- `listing.png` — listing detail or `/listings`
- `search-map.png` — BNHub search / map
- `dashboard.png` — host or demo dashboard

Run dev server + `python3 scripts/build_bnhub_premium_deck.py --capture` to auto-capture (Chrome required).
