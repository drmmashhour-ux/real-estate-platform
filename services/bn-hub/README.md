# BNHub Service

Short-term rental module: listings, booking, payments, reviews. Implemented in:

- **apps/web-app/lib/bnhub** — listings, booking, reviews
- **apps/web-app/app/api/bnhub** — API routes
- **apps/web-app/app/bnhub** — pages and host dashboard

See [BNHUB-BUSINESS-MODEL](../../docs/BNHUB-BUSINESS-MODEL.md).
